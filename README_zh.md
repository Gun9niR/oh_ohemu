## 介绍

该仓库下主要用作OpenHarmony社区开发的，而非某个厂商单独开发的典型产品样例代码
托管，主要包括类似QEMU的图形、软总线等特性产品的开发。

## 软件架构

支持QEMU的ARMv7-a架构的virt平台的图形显示产品样例，支持RISC-V架构的virt平台的
样例，以及Cortex-M4架构的样例。
